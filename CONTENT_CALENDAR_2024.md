## SEO Checklist for Each Post

- [ ] Primary keyword in title (front-loaded)
- [ ] Meta description 150-160 characters with keyword
- [ ] H1 tag contains primary keyword
- [ ] H2/H3 subheadings with related keywords
- [ ] Keyword density 1-2% (natural placement)
- [ ] 2-3 internal links to relevant pages
- [ ] 1-2 external links to authoritative sources
- [ ] Featured image optimized (under 200KB)
- [ ] Alt text on all images
- [ ] URL slug contains keyword
- [ ] FAQ schema markup (when applicable)
- [ ] Mobile-friendly formatting
- [ ] Read time 6-8 minutes
